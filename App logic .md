# **PART 2: DEVELOPER LOGIC (PSEUDO-CODE LEVEL)**

This is simplified logic structure.

---

## **DAILY SCORE CALCULATION**

Initialize:  
subconscious\_score \= 0  
conscious\_score \= 0  
results\_score \= 0  
total\_score \= 0

---

### **SUBCONSCIOUS LOGIC**

For each subconscious\_activity\_completed:  
    subconscious\_score \+= activity\_points

If subconscious\_score \> 40:  
    subconscious\_score \= 40

Minimum required:

If completed\_subconscious\_count \>= 2:  
    subconscious\_requirement\_met \= true  
Else:  
    subconscious\_requirement\_met \= false

---

### **CONSCIOUS LOGIC**

For each conscious\_activity\_completed:  
    conscious\_score \+= activity\_points

If conscious\_score \> 45:  
    conscious\_score \= 45

Minimum required:

If completed\_conscious\_count \>= 4:  
    conscious\_requirement\_met \= true  
Else:  
    conscious\_requirement\_met \= false

---

### **RESULTS LOGIC**

results\_score \+= (hot\_leads \* 5\)  
results\_score \+= (deals\_closed \* 20\)

If results\_score \> 15:  
    results\_score \= 15

---

### **FINAL DAILY SCORE**

total\_score \= subconscious\_score \+ conscious\_score \+ results\_score

If total\_score \> 100:  
    total\_score \= 100

---

### **STREAK LOGIC**

If subconscious\_requirement\_met AND conscious\_requirement\_met:  
    daily\_streak \+= 1  
Else:  
    daily\_streak \= 0

---

### **WEEKLY SCORE**

weekly\_average \= sum(daily\_scores) / days\_logged

# **PART 3: LEADERBOARD WITHOUT DEMOTIVATION**

This is where most apps fail.

If you rank purely by income:  
80% users quit.

If you rank by raw activity:  
New users feel crushed.

We use layered leaderboard logic.

---

# **LEADERBOARD MODEL: 3 CATEGORIES**

Instead of 1 ranking list, we use 3 rotating categories weekly:

---

### **1️⃣ Consistency Leaders (Primary Board)**

Ranked by:

weekly\_average\_score

This rewards discipline, not money.

Title:

“Top Operators – This Week”

---

### **2️⃣ Momentum Climbers**

Ranked by:

(current\_week\_average \- last\_week\_average)

This rewards improvement.

Even small users can win here.

Title:

“Biggest Improvers”

---

### **3️⃣ Deal Makers**

Ranked by:

deals\_closed\_this\_week

NOT commission value.

Keeps it clean.

---

# **PROTECTIVE PSYCHOLOGY**

We never show:

“You are rank 462”

Instead:

Show:

“You’re in Top 35% this week.”

OR

“You’re above your 30-day average.”

OR

“You’re 8 points away from Top 10%.”

This creates aspiration without shame.

---

# **OPTIONAL ADVANCED FEATURE**

Micro-Groups:

Users grouped into:

* New (\< 3 deals/month)  
* Growing (3–8 deals)  
* Elite (8+ deals)

Leaderboards are within cohort only.

This avoids beginner vs monster comparison.
